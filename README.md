# Accountabili-Buddy API

Accountabili-Buddy API provides to the developer access to user profile, teams, events, and gives the ability create new users, teams and events.

## Endpoints
- GET /users
- POST /users
- GET /users/:id
- POST /token
- GET /teams
- POST /teams  
- GET /teams/:id
- POST /teams/:id
- PUT /teams/:id
- DELETE /teams/:id
- GET /events
- POST /events

## API URL
https://bilibuddy-api.herokuapp.com/
